# Quick Alternative Hosting Options

## Option 1: Google Drive (2 minutes)
1. Upload `Parcel-Tools-Setup-2.0.0.exe` to Google Drive
2. Right-click → "Get link" → "Anyone with the link"
3. Copy the FILE_ID from the link
4. Use this URL format:
   ```
   https://drive.google.com/uc?export=download&id=FILE_ID
   ```

## Option 2: Dropbox (2 minutes)
1. Upload file to Dropbox
2. Click "Share" → "Create link"
3. Copy the link
4. **Change** `?dl=0` to `?dl=1` at the end
5. That's your download link!

## Option 3: MediaFire (Free, No Login Required)
1. Go to mediafire.com
2. Click "Upload"
3. Upload your .exe file
4. Get the direct download link
5. Use that link in your website

## Option 4: File.io (Quick Test)
1. Go to file.io
2. Upload file
3. Get temporary download link (works for testing)
