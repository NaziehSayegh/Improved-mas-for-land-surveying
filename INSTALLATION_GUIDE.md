## ‚ùì Troubleshooting

### "File is dangerous" Warning
**This is normal for new software!**
- Click "Keep" or "Run anyway"
- The software is safe - it's just not yet recognized by Windows SmartScreen

### Cannot Open .exe File
- Make sure file extension is `.exe` not `.zip`
- Right-click ‚Üí Properties ‚Üí Unblock ‚Üí Apply ‚Üí OK
- Then run the installer

### Installation Fails
- **Run as Administrator:**
  - Right-click the installer
  - Select "Run as administrator"
- Check antivirus isn't blocking it
- Make sure you have 500MB free disk space

### App Won't Start After Installation
- Check Windows Defender hasn't quarantined it
- Add exception in antivirus software
- Reinstall the application
