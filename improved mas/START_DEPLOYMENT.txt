TROUBLESHOOTING
========================================

Build fails?
- Check Node.js: node --version (need v16+)
- Check Python: python --version (need 3.8+)
- Delete node_modules, run: npm install

Website not working?
- Test locally: Double-click index.html
- Check download links are updated
- Wait a few minutes after deploying

Need help?
